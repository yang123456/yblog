# yblog 项目源码

## 运行项目

### 安装包
```shell
pnpm i
```

### 本地运行

```shell
pnpm run docs:dev
```

打开：[http://localhost:5174/yblog/](http://localhost:5174/yblog/)

### 项目构建

```shell
pnpm run docs:build
```
